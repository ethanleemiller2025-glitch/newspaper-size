# Newspaper Advertisement Size Converter

This is website to convert newspaper advertisement sizes (in column centimeters, cc) into measurable units (centimeters and inches) and preview the ad size visually.

## Features
- Enter ad size in column centimeters (cc)
- Choose aspect ratio (custom, 1:1, 2:1, 1:2, 4:3, 3:4)
- Enter number of columns and column width
- Converts and displays dimensions in cm and inches
- Visual preview of the ad size

## Usage
1. Open `index.html` in your browser.
2. Enter the ad size in cc.
3. Select an aspect ratio or use custom columns.
4. Enter the number of columns and column width (cm).
5. Click "Convert" to see the dimensions and preview.

## Example
- 50cc, 2 columns, 3.5cm column width, aspect ratio 2:1
- Result: 7.00cm × 25.00cm (2.76in × 9.84in)
